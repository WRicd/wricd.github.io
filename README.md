# wricd.github.io

Portfolio site of Richard Wang.

## Structure

- `index.html`: Three.js starfield homepage and project entry.
- `js/projects.json`: Shared project data for the homepage, list page, and detail template.
- `projects/index.html`: Project grid generated from JSON.
- `projects/template.html`: Project detail template loaded by `?id=...`.
- `about/index.html`: About page placeholder.
- `resume/index.html`: Resume page placeholder.

Run a simple static server from the repository root to preview locally.
